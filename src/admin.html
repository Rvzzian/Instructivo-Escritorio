Uso del Software

A continuación se detallan las funcionalidades disponibles para el perfil de administrador:

  2. Reportes
  3. Gestión de Usuarios

  • 1. Inicio (HOME)

    Al iniciar sesión como administrador, el sistema mostrará el menú principal con las siguientes opciones: Inicio, Reportes, Usuarios e Historial.

  • Visualización de Métricas

    La sección Inicio presenta métricas del sistema mediante gráficos interactivos que facilitan el análisis de datos. Estos visualizadores permiten monitorear el estado general de la plataforma.

  • Las métricas se organizan de forma intuitiva para una rápida interpretación. Puede seleccionar entre diferentes conjuntos de datos según sus necesidades específicas de análisis.

  • 2. Gestión de Reportes

    En esta sección podrá consultar todos los reportes generados por los usuarios sobre documentos perdidos. La información se presenta en formato de tabla con tarjetas individuales e incluye un sistema de filtrado avanzado.

  • Búsqueda de Reportes

    Utilice la barra de búsqueda para localizar reportes específicos ingresando el número de identificación asociado.

  • Navegación entre Resultados

    El sistema muestra un máximo de 10 resultados por página. Utilice los controles de paginación para navegar entre los diferentes conjuntos de resultados.

  • Filtrado por Estado

    Puede filtrar los reportes según su estado actual. Los estados disponibles son:

    • En revisión
    • Identidad confirmada
    • No aprobado
    • Aprobado
    • Entregado
    • Archivado

  • Detalle de Reportes

    Para acceder a información completa de un reporte, haga clic en el botón "Ver detalles". Esto abrirá una ventana con todos los datos asociados al caso.

  • Ventana de Detalles

    Esta vista ampliada muestra toda la información relevante del reporte, incluyendo:

    • ID del reporte
    • Fecha de creación
    • Ubicación reportada
    • Detalles adicionales
    • ID del usuario
    • Estado actual
    • Información del propietario
    • Número y tipo de documento

  • Visualización de Imágenes

    Para ampliar cualquier imagen adjunta, realice doble clic sobre ella. Para cerrar la vista ampliada, haga clic en la "X" ubicada en la esquina superior derecha.


  • Actualización de Estados

    Desde esta ventana puede modificar el estado del reporte. Antes de realizar cualquier cambio, asegúrese de validar cuidadosamente la información.

    Confirmación de Cambios

    Para guardar los cambios realizados, haga clic en el botón "Guardar". El sistema mostrará una confirmación cuando la operación se complete exitosamente.

    Para cerrar la ventana sin guardar cambios, utilice el botón "Cancelar".

  • 3. Gestión de Usuarios

    Esta sección permite administrar todos los usuarios registrados en el sistema, incluyendo la creación, modificación y consulta de perfiles.


    La navegación entre páginas de resultados se realiza mediante los controles inferiores, mostrando 10 registros por página.

  • Búsqueda de Usuarios

    Localice usuarios específicos ingresando su nombre en la barra de búsqueda y presionando el ícono de lupa.


    Ejemplo de búsqueda:

  • Detalle de Usuario

    Para consultar información detallada de un usuario, haga clic en el enlace "Ver" correspondiente al registro deseado.

  • Ventana de Detalles de Usuario

    Esta vista muestra toda la información del usuario seleccionado y permite modificar su estado y tipo de perfil.


    Modificación de Estado

    Puede cambiar el estado del usuario entre "Activo" e "Inactivo" según las políticas de administración. Seleccione la nueva opción del menú desplegable.


    Cambio de Tipo de Usuario

    Los tipos de perfil disponibles son: Usuario estándar, Soporte y Administrador. Seleccione la nueva categoría según las necesidades operativas.


    Confirmación de Cambios

    Para aplicar las modificaciones, haga clic en "Guardar". El sistema notificará el éxito de la operación.


    Mensaje de confirmación:


    Para descartar cambios y cerrar la ventana, utilice el botón "Cancelar".

  • Creación de Nuevos Usuarios

    Para registrar un nuevo usuario en el sistema, haga clic en el botón "Crear". Esto abrirá un formulario de registro.


    Formulario de Registro

    Complete todos los campos requeridos para crear una nueva cuenta. Esta función está diseñada principalmente para la creación de perfiles administrativos (Admin y Soporte), aunque también permite registrar usuarios estándar.

    Finalización del Registro

    Al completar el formulario, haga clic en "Crear Usuario". El sistema confirmará el registro exitoso.

  • 4. Historial de Auditoría

    Esta sección registra todas las acciones relevantes realizadas en el sistema, mostrando:

    • ID de registro
    • Usuario responsable
    • Acción realizada
    • Modelo afectado
    • ID del objeto
    • Fecha y hora
    • Opciones de visualización

  • Sistema de Filtrado

    El historial cuenta con potentes herramientas de filtrado que permiten obtener información específica según diversos criterios.

  • Consulta Detallada

    Para obtener más información sobre un registro específico:

    1. Localice la columna "Vista"
    2. Haga clic en el botón "Ver" correspondiente al registro de interés

  • Opciones de Filtrado Avanzado

    El sistema permite filtrar el historial de tres formas diferentes:

  • Por Usuario: Seleccione un nombre de usuario específico para visualizar todas sus acciones registradas.

  • Por Fecha: Especifique un rango de fechas para limitar los resultados a un período determinado.

  • Por Tipo de Operación: Filtre entre las categorías disponibles: Todos, Usuario, Reporte o PQRS.

  • Los resultados se muestran en conjuntos de 20 registros. Utilice los controles de paginación para navegar entre las diferentes páginas de resultados.

  • 5. Salir del Sistema

    Para cerrar sesión y salir de la aplicación, haga clic en el botón "Salir" ubicado en el menú principal.

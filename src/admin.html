Perfil administrador

Uso del Software

Aquí se describen las funcionalidades principales del perfil admin:

  • ADMINISTRADOR

    1. Inicio (HOME)

    Una vez ingreses como administrador, se desplegarán varias opciones en el menú principal: HOME, REPORTES, USUARIO e HISTORIAL.

  • Visualización de Métricas

    En la sección HOME podrás consultar diversas métricas del sistema a través de gráficas dinámicas.

  • Las métricas se muestran de manera visual para facilitar su análisis.

    Puedes seleccionar la métrica que deseas consultar según tu necesidad.

  • Filtros

    Además, puedes aplicar filtros para ajustar la información visualizada, lo cual te permitirá obtener datos más específicos de acuerdo al criterio seleccionado.

  • 2. Historial (HISTORIAL)

    Al acceder a la opción HISTORIAL, encontrarás un listado detallado que incluye:

    ID, Usuario, Accion, Modelo, Id_objeto, Fecha, Vista

  • Consulta del Historial

    Para consultar el historial reciente

    1. Dirigete a la columna Vista.

    2. Haz clic en el perfil que deseas consultar.

  • Opciones de Filtro

    el historial puede filtrarse de tres maneras:

  • Por Usuario: selecciona el nombre del usuario para ver sus acciones registradas.

  • Por Fecha: elige unn rango de fechas específicos para limitar la consulta.

  • Por Tipo: filtra por el tipo de operacion realizada( todos, usuario, reporte, pqrs).
